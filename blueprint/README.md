# Entrance Question — Blueprint

This folder contains a single source-of-truth document, [`BLUEPRINT.md`](./BLUEPRINT.md),
that fully specifies the **Entrance Question** app (a Next.js MCQ practice + admin platform).
It lives inside the project (`/blueprint`) so it is always available alongside the code.

## What this is for

Hand `BLUEPRINT.md` to any AI coding agent (or developer) and they should be able to rebuild the
entire application from scratch — same stack, data model, features, and UI behavior — without
needing access to the original codebase.

## Keeping it current

`BLUEPRINT.md` is kept in sync with the app **automatically**. Whenever a feature is added or
changed in the app, the corresponding section here is updated and the [changelog](./BLUEPRINT.md#19-changelog)
gets a new entry.
